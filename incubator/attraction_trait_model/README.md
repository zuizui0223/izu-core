# Attraction and mating-system model incubator

This directory is reserved for the former attraction-trait model: nectar-guide expression, flower size, delayed-selfing geometry, selfing ability, neutral diversity, and spatial position.

It is **not yet an active inference engine** and must not be used to make a field claim. Promotion requires all four:

```text
1. one explicit biological hypothesis;
2. one declared life cycle and parameter interpretation;
3. one observable measurement set linked to the model;
4. one falsification or discrimination target.
```

The intended first use is likely to distinguish a pollinator-mediated attraction pathway from selfing / delayed-selfing compensation inside the local reproduction component F. That question is separate from the base F-versus-E protocol and should be introduced only when the planned field measurements make it testable.
