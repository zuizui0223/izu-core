# Guide/spot photo trait-review handout

You are reviewing flower photographs without island names, coordinates, or
observation pages. Please do not seek those metadata while scoring.

Each line represents one iNaturalist observation. Several photo URLs in the
same line are alternative views of that same observation; do not treat them as
separate flowers.

## Complete these fields

- `focal_taxon_consistent`: `yes`, `no`, or `unclear` from visible flower form.
- `inner_corolla_visibility`: `adequate`, `partial`, or `not_visible`.
- `flower_open_stage`: `open`, `not_open`, or `unclear`.
- `image_comparable`: `yes` only when focus, angle, illumination, and visible
  inner corolla permit comparison with the other blinded observations.
- `guide_ordinal_0_to_3`: complete only when the previous three gates support a
  score.
- `trait_review_status`: `accepted` or `excluded`.
- `exclusion_reason`: required when excluded.

## Ordinal score

| score | inner-corolla guide/spot expression |
|---:|---|
| 0 | no or negligible visible expression |
| 1 | weak expression |
| 2 | moderate expression |
| 3 | strong expression |

Use the full set of linked photos for one observation, but provide one score per
row. Exclude a row rather than estimating a score when angle, flower stage,
lighting, image editing, or focus makes the inner-corolla pattern ambiguous.

Do not infer abundance, evolutionary history, pollinator type, or island
membership from these photographs. This task is only a blinded ordinal screen
of visible floral pattern.
