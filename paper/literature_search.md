# Literature and public-data search record

## Current status — 2026-07-03

This is an active, audited search workstream. It is **not** a claim that the
available public literature has been exhausted. The 156-species entomophilous
table is a pilot analysis cohort; its reported parent universe is broader and is
being rebuilt from source-logged GBIF queries.

A search hit is only a discovery lead. A source enters the evidence table only
after its taxonomy, geography, comparison design, wild/cultivated context,
trait definition, and page/table/figure are checked. Occurrence and photo
availability are not trait-response observations.

## Search lanes

1. Direct mainland–island or ordered-island floral and reproductive comparisons.
2. Taxon-name expansion using accepted names, synonyms, Japanese names and older
   combinations.
3. Pollination functional-group evidence needed for the mechanism moderator.
4. Blinded photo eligibility review, kept separate from numeric effects.
5. Occurrence and environmental data for coverage and alternative explanations.

Each retained or excluded source must have: exact query, retrieval date, source
identifier, source page/table/figure, comparison unit, inclusion decision and
exclusion reason. The schema is in `paper/evidence_screening/`.

## Explicit boundaries

- The whole U0 parent universe has not yet received source-level Japanese and
  English screening.
- Generalist negative-control effects are not estimated.
- Comparative taxa, named island forms and Web/flora descriptions are not
  independent within-species geographic replicates.
- Cross-lineage effect pooling, a shared breakpoint claim and a pollination-group
  interaction remain pending source recovery.

## Completion condition

A literature search can be called complete only after the U0 parent table is
source-locked; each taxon has an accepted-name/synonym query record; Japanese and
English source lanes have logged hits and exclusions; and every retained
observation has an explicit synthesis role plus a source page/table/figure.
